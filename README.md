# freepik

Repo planistyczne pod aplikację desktop/CLI do automatycznego wyszukiwania i pobierania stock video z Freepik na podstawie `stockplan.json`.

## Co jest w repo
- `docs/STATUS.md` — bieżący stan i następny krok.
- `docs/ARCHITECTURE.md` — moduły, przepływ danych, output.
- `docs/WORKFLOW_FOR_AI.md` — twardy protokół pracy dla AI.
- `docs/ROADMAP.md` — fazy 0–8.
- `docs/TESTING.md` — strategia testów.
- `docs/DECISIONS/` — ADR-y.
- `docs/prompts/` — gotowe prompty:
  - ChatGPT → generowanie `stockplan.json`
  - Codex/Claude → implementacja aplikacji fazami
- `.github/` — template PR, issue templates, CI skeleton.

## Start
1. Przeczytaj `docs/STATUS.md`.
2. Przeczytaj `docs/ARCHITECTURE.md`.
3. Realizuj roadmapę od Fazy 0.
